## 🎯 Что такое Worka?

**Worka** — это desktop-приложение, которое автоматически отслеживает, как вы проводите время за компьютером. Никаких ручных таймеров и забытых записей — только пассивный сбор метаданных о вашей активности.

### Ключевые возможности

- 🔍 **Автоматический трекинг приложений** — Worka запоминает, в каких программах вы работаете
- ⏱️ **Учёт активного времени и простоя** — различает работу и паузы
- ⌨️ **Счётчики ввода** — отслеживает количество нажатий клавиш и кликов мыши (без сохранения текста)
- 📊 **Наглядные отчёты** — KPI, графики и таймлайны за день и неделю
- 🔒 **Приватность** — все данные хранятся локально на вашем устройстве
- 🎛️ **Гибкие настройки** — интервал опроса, порог idle, включение/отключение трекинга

## ✨ Возможности

### 🖱️ Работа в трее

Worka работает в системном трее Windows:

- **Левый клик по иконке** — открывается меню с пунктами:
  - **Открыть** — показать дашборд приложения
  - **Выйти** — полностью закрыть приложение (сбор данных останавливается)

- **Закрытие окна (крестик)** — приложение сворачивается в трей (не закрывается)
- **Данные собираются** — пока приложение запущено, трекинг работает в фоне
- **При повторном запуске** — открывается существующее окно (не создаётся копия)

### 📈 Дашборд «Сегодня»

Смотрите статистику за текущий день:
- **KPI**: активное время, время простоя, нажатия клавиш, клики мыши
- **Топ приложений** — какие программы занимали больше всего времени
- **График интенсивности ввода** — пики и спады активности в течение дня

### 📅 Дашборд «Неделя»

Анализируйте продуктивность по дням:
- **Активное/idle время по дням** — выявляйте закономерности
- **Топ приложений за неделю** — что доминирует в вашей работе
- **Суммарный ввод** — общая активность за период

### 🕐 Дашборд «Таймлайн»

Детальный просмотр дня:
- **Лента активности** — последовательность приложений и периодов простоя
- **Дорожка ввода** (опционально) — интенсивность работы с клавиатурой и мышью

### ⚙️ Настройки

Полный контроль над трекингом:
- **Интервал опроса** — как часто собирать данные (по умолчанию 10 секунд)
- **Порог idle** — через сколько секунд бездействия считать вас неактивным (120–300 секунд)
- **Приватность** — включение/отключение трекинга заголовков окон и ввода
- **Пауза/Продолжить** — временная остановка сбора данных
- **Автозапуск** — начинать трекинг при старте приложения

## 🛡️ Конфиденциальность

Worka разработан с принципом **минимального вторжения**:

- ❌ **Не сохраняет** текст набранных символов
- ❌ **Не делает** скриншоты экрана
- ❌ **Не передаёт** данные на внешние серверы
- ✅ **Хранит всё локально** в SQLite-базе на вашем устройстве
- ✅ **Только метаданные**: название приложения, время активности, количество нажатий

## 📝 Управление версией

Версия указывается **только в `package.json`** и автоматически синхронизируется:

- `src-tauri/Cargo.toml`: `version = { workspace = true }`
- `src-tauri/tauri.conf.json`: `"version": "../package.json"`

Для изменения версии достаточно обновить `package.json` — остальные файлы подхватят автоматически.
