# Функционал приложения (Windows 10)

## Уровень 1 — MVP (делаем сразу)

### Основное поведение
- Приложение работает в трее
- При закрытии окна — остаётся в трее
- Выход только через пункт «Выход» в меню трея
- Возможность поставить сбор на паузу

### Сбор данных
- Опрос каждые N секунд (по умолчанию 10)
- Определение:
  - активного приложения (process name / exe path)
  - заголовка активного окна (опционально)
  - статуса idle (нет ввода N секунд)
- Сохранение данных в SQLite

### Дашборд (3 независимых канала)

#### Канал 1 — Время (основной)
Что трекаем:
- активное приложение (exe / путь)
- статус активности: Active / Idle (по порогу idle)
- (опционально) заголовок активного окна

Как показываем:
- KPI: активное время, время простоя
- Топ приложений по времени (бар + таблица)
- Таймлайн дня: сегменты «приложение / idle» по времени

#### Канал 2 — Ввод (опционально, только количество)
Что трекаем (без сохранения текста и без логирования конкретных клавиш):
- отдельно счётчик **клавиатуры** (кол-во нажатий)
- отдельно счётчик **мыши** (кол-во кликов)
- (опционально) кол-во прокруток колеса

Как храним:
- агрегаты по времени (например по минутам), без сырых событий

Как показываем:
- KPI: нажатия клавиш за день, клики мыши за день
- График «интенсивность ввода» по времени (линия/столбцы): клавиатура и мышь отдельными рядами
- (опционально) “интенсивность” поверх таймлайна (вторая дорожка)

#### Канал 3 — Метаданные (настройки и управление)
- Настройки: интервал опроса, порог idle
- Переключатели приватности: трекать заголовки окон (да/нет), считать ввод (да/нет)
- Управление: пауза/продолжить сбор, автозапуск
- Экспорт/бэкап (позже)

#### Экраны/вкладки UI
- **Сегодня**: KPI + топ приложений + интенсивность ввода
- **Неделя**: активное/idle по дням + топ приложений за неделю + суммарный ввод
- **Таймлайн**: детальный день (приложения/idle) + (опционально) дорожка ввода
- **Настройки**: параметры сбора + приватность + управление

### Настройки
- Интервал опроса
- Порог idle (например 120–300 секунд)
- Включение/отключение трекинга заголовков окон
- Автозапуск (по желанию)
